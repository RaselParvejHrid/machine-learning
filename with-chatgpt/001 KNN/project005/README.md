## Problem Statement

Before we leave KNN, I'd like you to build one polished project.

### Suggested Dataset

Start with one of these built into SciKit-Learn:

- Iris
- Wine (I chose this)
- Breast Cancer Wisconsin

After that, try a Kaggle dataset with mostly numerical features.

### Requirements

Build a complete workflow:

* Exploratory Data Analysis (brief, using pandas)
* Train/test split
* Pipeline
* Appropriate preprocessing
* GridSearchCV
* Evaluation with:
  * Accuracy
  * Confusion Matrix
  * Classification Report
* Discussion of:
  * Best hyperparameters
  * Why those hyperparameters may have worked
  * Strengths and weaknesses of KNN on this dataset
