## Problem Statement

Using the built-in Iris dataset:

* Load the dataset.

* Split it into training and testing sets.
* Build a `Pipeline` with:

  * `StandardScaler`
  * `KNeighborsClassifier`
* Train the pipeline.

* Predict on the test set.
* Report the test accuracy.

**Do not** use `GridSearchCV` yet—we'll learn that properly in a later session.

---

This project is intentionally small. The goal isn't to build the world's best classifier—it's to establish a professional workflow that we'll reuse for every algorithm we study from here onward.
