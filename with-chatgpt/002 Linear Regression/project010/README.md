# Project 10 | Multiple Linear Regression Pipeline

## Problem Statement

Use the California Housing dataset (if available in your environment) or another tabular regression dataset.

Your workflow should include:

1. Identify:
   * numerical features,
   * categorical features (if the dataset has any).
2. Build a `ColumnTransformer`.
3. Build a `Pipeline` with:
   * preprocessing,
   * LinearRegression.
4. Evaluate using:
   * MAE,
   * RMSE,
   * R².
5. Inspect the learned coefficients and match them to the transformed feature names.
6. In your README, answer:
   1. Which features appear to contribute positively?
   2. Which contribute negatively?
   3. Which coefficients are hardest to interpret, and why?
